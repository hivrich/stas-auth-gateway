Тест страницы авторизации

URL: https://intervals.stas.run/gw/oauth/authorize?client_id=chatgpt-actions

Инструкции:

  1. Откройте URL выше в браузере
  2. Введите любой STAS ID (например: 12345)
  3. Откройте консоль браузера (F12 → Console)
  4. Нажмите кнопку "Войти"
  5. Посмотрите логи в консоли

Что должно происходить:

При нажатии кнопки в консоли должны появиться логи:
handleLogin called
STAS ID: [введенный ID]
redirectUri: [пусто или значение]
Redirecting to: [URL для редиректа]

Если кнопка не работает:

  • Проверьте, что JavaScript включен в браузере
  • Посмотрите ошибки в консоли браузера (красные сообщения)
  • Попробуйте другой браузер
  • Попробуйте отключить блокировщик рекламы

Альтернативный тест:

Если основная страница не работает, попробуйте:

https://intervals.stas.run/gw/oauth/token?user_id=12345&client_id=chatgpt-actions

Это должно сразу сделать редирект на chat.openai.com
